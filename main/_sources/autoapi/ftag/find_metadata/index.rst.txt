      Fetch task details from the BigPanDA JSON API.

      :param taskid: The BigPanDA Task ID.
      :type taskid: str

      :returns: The first task record from the API response, or ``None`` if unavailable.
      :rtype: dict | None



   .. py:method:: _extract_container(info: dict) -> str | None

      Find the MC container name within the task information.

      :param info: The task information dictionary from BigPanDA.
      :type info: dict

      :returns: The container name (e.g., 'mc16_13TeV...'), or ``None`` if not found.
      :rtype: str | None



   .. py:method:: _parse_info(container: str) -> tuple[int, str, str] | None

      Parse DSID, etag, and campaign from a container name.

      :param container: The full MC container name.
      :type container: str

      :returns: A tuple of (DSID, etag, campaign), or ``None`` if parsing fails.
                Note: 'mc20' is automatically mapped to 'mc16'.
      :rtype: tuple[int, str, str] | None



   .. py:method:: _download_db(campaign: str) -> str

      Download the PMG database for a specific campaign.

      :param campaign: The MC campaign name (e.g., 'mc16').
      :type campaign: str

      :returns: The local path to the downloaded database file.
      :rtype: str



   .. py:method:: _query_xsecdb(campaign: str, dsid: int, etag: str) -> dict | None

      Search the PMG database for metadata matching a DSID and etag.

      :param campaign: The MC campaign name.
      :type campaign: str
      :param dsid: Dataset ID.
      :type dsid: int
      :param etag: The AMI tag (e.g., 'e1234').
      :type etag: str

      :returns: Dictionary containing 'cross_section_pb', 'genFiltEff', and 'kfactor',
                or ``None`` if no matching record is found.
      :rtype: dict | None



   .. py:method:: inject_metadata() -> None

      Execute the full workflow to inject metadata into the HDF5 file.

      This method coordinates Task ID extraction, API fetching, DB querying,
      and final HDF5 attribute writing. Metadata is stored in a group
      named ``metadata/{dsid}``.
